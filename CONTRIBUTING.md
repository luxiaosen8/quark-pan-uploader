# Contributing

感谢你对本项目的关注。

## 基本原则

- 保持修改范围聚焦，避免顺手改动无关代码
- 提交前确认未包含本地日志、输出目录、构建产物或敏感配置
- 如需提交示例数据，请确保全部完成脱敏

## 提交建议

1. 新建分支进行修改
2. 完成自检后再提交
3. 在提交说明中明确变更目的与范围
4. 如涉及 UI 或行为变化，请同步更新 README 或相关公开文档

## 安全与隐私

贡献代码时请不要提交以下内容：

- 真实 Cookie / Token
- `.env` 文件
- 本地绝对路径或用户目录信息
- 调试日志、输出结果、打包产物

如需报告安全问题，请优先阅读 [SECURITY.md](./SECURITY.md)。
